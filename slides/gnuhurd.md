##  GNU/Hurd


- Hurd
    - Kernel developed by the GNU project
    - Meant to be a replacement for Linux

- Built in C
- Implements in true Microkernel

### Classification

| GNU/Hurd |
|----------------------------|
| Interface type:        | CLI/GUI |
| Internal Architecture: | Microkernel  |
| Operating Mode:        | Multi-user, Multitasking  |
